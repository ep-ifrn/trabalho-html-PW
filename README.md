# trabalho-html-PW

Aqui está o projeto de criação de site do trabalho de Padrões web

Neste site, a proposta é seguir as exigências do professor e criar uma página demonstrando o que cada tag executa durante o código, para que o estudante aprenda junto com a tag durante a leitura. Aqui estão as exigências:

- Estrutura Semântica: Uso correto de <h1>, <h2>, e outros níveis de título para organizar o conteúdo.
- Parágrafo : Pelo menos dois parágrafos sobre o conteúdo ...
- Listas: Pelo menos uma lista (<ul> ou <ol>) contendo itens (<li>).
- Tabela : Uma tabela com no mínimo 4 colunas e 3 linhas ...
- Imagens: Inserção de uma imagem com o atributo alt (explicar por que o alt é importante para a acessibilidade).
- Links: Pelo menos um link (<a>) apontando para um site externo.

Boa leitura de código.
