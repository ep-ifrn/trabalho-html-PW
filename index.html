Explicação de HTML básico

Aqui ensinaremos aos curiosos como funcionam as tags na linguagem de marcação HTML com exemplos práticos!

Sumário:

  1. Níveis de título;
  2. Parágrafos;
  3. Listas;
  4. Tabela;
  5. Imagem e uso de alt;
  6. Links.

Parte 1 - Níveis de Títulos

Imagine que você está escrevendo em um pedaço de papel um texto. Ele possui título, parágrafos, imagens, entre outros. No HTML os títulos são feitos usando os comandos <h1>, <h2>, <h3>, <h4>, <h5> e <h6>. Utilizando-os desta maneira e repetindo a tag com uma barra "/" após o símbolo "<" como nos exemplos abaixo:

<h1> Hello, world! </h1>

<h2> Hello, world! </h2>

<h3> Hello, world! </h3>

Parte 2 - Parágrafos

De forma semelhante, ao escrever um parágrafo para detalhar coisas no site usando HTML, deve-se inserir entre o texto a tag "<p>" e ao final "</p>", como no exemplo abaixo:

<h1> Hello, world! </h1>

<p> Tradução: Olá, mundo! </p>

Parte 3 - Listas

Em HTML, existe 2 tipos de listas. é importantíssiomo aprendê-las para que a estrutura de seu site seja precisa e organizada, são elas o <ol> e <ul>, eles servem respectivamente para criar listas enumeradas, onde a ordem importa, e listas não enumeradas, onde a ordem não importa. E ao decorrer da lista deve-se colocar a tag <li> para criar a sequência. Por exemplo:

<h1> Lista de tarefas (ordem relevante)</h1>

<ol>
<li>Ir ao trabalho de 7h</li>
<li>Comprar roupas de 18h</li>
<li>Lavar o carro de 19h</li>
</ol>

Agora um exemplo de lista que não importa a ordem:

<h1> Lista de tarefas (ordem irrelevante)</h1>

<ul>
<li>Jogar bola</li>
<li>jogar vôlei na praia</li>
<li>Tomar um café sem açúcar</li>
</ul>

Parte 4 - Tabela

Tabelas em HTML demandam um pouquinho mais de trabalho, mas não são um bicho de 7 cabeças. O processo é bem simples, nele utilizaremos o <table> para indicar que estamos criando uma tabela, dentro dela colocaremos a tag <tr> para indicar as linhas e por fim a tag <td> para inserir os dados na tabela. Aqui está uma demonstração: 

<h1>Tabela<h1>

<table>

<tr>

<td>informação A1</td>

<td>informação A2</td>

<td>informação A3</td>

</tr>

</table>

Durante a demonstração na web, aparecerá desta maneira:

A1 A2 A3

Sabendo dessa informação, basta agora inserirmos as colunas repetindo o <tr> para criarmos uma nova linha e inserindo o <td> e assim por diante. Aqui está um exemplo de uma tabela de 4 colunas e 3 linhas sem muitos detalhes de forma básica:

A1 A2 A3 A4
B1 B2 B3 B4
C1 C2 C3 C4

Para deixar a tabela mais semelhante como tabelas profissionais com células mais estruturadas usaremos uma tag chamada <style> que definirá o estilo da célula e após inserí-la, pediremos para colocar na tabela um limite separando célula por célula usando a tag "border", para definir uma fronteira, logo na linha de baixo podemos definir a largura. Exemplo: 

<h1>Tabela<h1>

<style>

table {

  witdth: 400px;

}

td {

  border: 1px solid black;

               padding:8px

}

</style>

<table>

<tr>

<td>informação A1</td>

<td>informação A2</td>

<td>informação A3</td>

<td>informação A4</td>

</tr>

</table>

<tr>

<td>informação B1</td>

<td>informação B2</td>

<td>informação B3</td>

<td>informação B4</td>

</tr>

<tr>

<td>informação C1</td>

<td>informação C2</td>

<td>informação C3</td>

<td>informação C4</td>

</tr>

visualmente ficará mais agradável:

A1 A2 A3 A4
B1 B2 B3 B4
C1 C2 C3 C4

Imagens e uso de alt

Para acrescentar uma imagem é muito simples, basta escrever <img>, é importante saber que ainda dentro dos símbolos maior e menor deve-se inserir o "src=" e "alt=", cada um com sua devida função. O "src=" tem a função de indicar onde está a imagem nos arquivos da máquina, enquanto o "alt=" possui a funcionalidade de deixar uma mensagem sobre a imagem, que é útil para pessoas com deficiência visual pois assim a máquina consegue transmitir a mensagem dizendo que ali há uma imagem. Uma ilustração de como ficaria no código seria essa: 

        <img src="insira_de_onde_vem_o_arquivo_aqui" alt="Imagem X">
